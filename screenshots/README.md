# Screenshots

PNG assets in this folder are used by the root [`README.md`](../README.md).

- **Regenerate:** from repo root, run `pnpm screenshots:readme` (requires [Playwright](https://playwright.dev/) Chromium — installed with `pnpm exec playwright install chromium`).
- **Source routes:** static, public pages under `apps/web/src/app/readme-preview/` (dashboard chrome without Clerk — see middleware matcher).

Replace any file here if you capture real signed-in sessions locally.
